# Customer Shopping Behavior Analysis

## 📌 Project Overview

This project analyzes customer shopping behavior data to uncover purchasing patterns, customer preferences, subscription trends, and revenue insights. The analysis combines SQL, Python, and Power BI to transform raw customer transaction data into actionable business intelligence.

### Objectives
- Analyze customer purchasing behavior.
- Identify top-performing products and categories.
- Compare spending patterns across customer segments.
- Evaluate the impact of discounts and subscriptions.
- Build an interactive dashboard for business decision-making.

---

## 📂 Dataset Information

### Dataset
**File:** `customer_shopping_behavior.csv`

### Records
- 3,900 customer transactions
- 18 attributes

### Features

| Column | Description |
|----------|-------------|
| Customer ID | Unique customer identifier |
| Age | Customer age |
| Gender | Male/Female |
| Item Purchased | Product purchased |
| Category | Product category |
| Purchase Amount (USD) | Transaction value |
| Location | Customer location |
| Size | Product size |
| Color | Product color |
| Season | Purchase season |
| Review Rating | Product rating |
| Subscription Status | Subscription status |
| Shipping Type | Delivery method |
| Discount Applied | Whether discount was used |
| Promo Code Used | Whether promo code was used |
| Previous Purchases | Number of previous purchases |
| Payment Method | Payment mode |
| Frequency of Purchases | Purchase frequency |

## 🛠 Data Preprocessing

The dataset was cleaned and prepared using Python.

### Steps Performed

- Checked data types and dataset structure.
- Identified missing values.
- Filled missing review ratings using:
  
```python
df.groupby('Category')['Review Rating'].transform('median')
```

- Standardized column names using snake_case.
- Performed exploratory data analysis (EDA).

## 📊 SQL Analysis

The following business questions were answered using PostgreSQL:

### 1. Revenue by Gender
- Compare total revenue generated by male and female customers.

### 2. High-Spending Discount Users
- Customers who used discounts but still spent above average.

### 3. Top Rated Products
- Products with highest average review ratings.

### 4. Shipping Comparison
- Average purchase amount between Standard and Express shipping.

### 5. Subscription Analysis
- Compare:
  - Total customers
  - Average spending
  - Total revenue
  between subscribers and non-subscribers.

### 6. Discount Utilization
- Products with highest discount usage percentage.

### 7. Customer Segmentation
Customers classified as:

| Segment | Criteria |
|----------|---------|
| New | 1 previous purchase |
| Returning | 2–10 previous purchases |
| Loyal | More than 10 previous purchases |

### 8. Top Products by Category
- Top 3 most purchased products within each category.

### 9. Repeat Buyer Subscription Analysis
- Whether repeat customers are more likely to subscribe.

### 10. Revenue by Age Group
- Revenue contribution across age segments.

## 🔍 Key Insights

### Customer Behavior
- Certain product categories receive significantly higher purchase volumes.
- Repeat customers contribute a substantial portion of revenue.

### Discounts
- Discounts increase purchase frequency but not necessarily customer loyalty.

### Subscription Trends
- Subscribers generate strong recurring revenue.
- Subscription status influences purchasing patterns.

### Product Performance
- Some products maintain consistently high customer ratings.
- Top-performing products vary across categories.
